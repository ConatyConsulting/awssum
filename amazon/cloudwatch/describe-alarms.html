DescribeAlarms (Open docs on AWS)

Params

Param Name Required Type Notes
Action required param
ActionPrefix optional param
AlarmNamePrefix optional param
AlarmNames optional param-data
MaxRecords optional param
NextToken optional param
StateValue optional param

Examples

List all alarms

This example shows how to list all alarms.

Example

cw.DescribeAlarms({}, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
cw.DescribeAlarms(function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});

List all alarms with a specific state

This example shows how to list all alarms with a specific state. Values of StateValue can be "OK", "ALARM" or "INSUFFICIENT_DATA".

Example

var params = {
    StateValue : 'INSUFFICIENT_DATA'
};
cw.DescribeAlarms(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
