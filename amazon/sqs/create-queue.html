CreateQueue (Open docs on AWS)

Params

Param Name Required Type Notes
Action required param
AttributeName optional param-array-set
AttributeValue optional param-array-set
QueueName required param

Examples

Create a SQS queue

This example shows how to create a queue to put messages onto.

Example

var params = {
    QueueName       : 'yourQueueName'
};
sqs.CreateQueue(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});

Create a SQS queue with specific attributes

This example shows how to create a queue to put messages onto. It shows how to adjust some default properties of a send SQS message onto this queue like the time (30 seconds) it will timeout before it is visible again for other SQS-processes.

Example

var params = {
    QueueName       : 'yourQueueName',
    AttributeName   : ['VisibilityTimeout'],
    AttributeValue  : [30]
};
sqs.CreateQueue(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
