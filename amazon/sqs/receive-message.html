Examples

Receive a message from a SQS queue

This example shows how to receive a message from a SQS queue to process and later execute SQS.DeleteMessage to clean up the SQS queue.

Example

var params = {
    QueueName : 'yourQueueName'
};
sqs.ReceiveMessage(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
