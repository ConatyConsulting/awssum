Examples

List the URL of a certain queue

This example shows how to list the URL of a certain queue. This can be useful in combination with the QueueOwnerAWSAccountId-property you can add to the params to retrieve other queues from other accounts to send messages to or process them.

Example

var params = {
    QueueName : 'yourQueueName'
}
sqs.GetQueueUrl(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
