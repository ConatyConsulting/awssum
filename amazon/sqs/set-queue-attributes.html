Examples

Set a specific attribute of a SQS queue.

This example shows how to set a specific attribute of a SQS queue. Valid attributes are VisibilityTimeout, Policy, MaximumMessageSize, MessageRetentionPeriod or DelaySeconds. No Array is allowed, so you need to update per attribute.

Example

var params = {
    QueueName       : 'yourQueueName',
    AttributeName   : 'DelaySeconds',
    AttributeValue  : 30
}
sqs.SetQueueAttributes(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
