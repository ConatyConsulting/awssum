Examples

Send a batch of messages onto a SQS queue

This example shows how to send a batch of messages (maximum of 10 and per message maximum of 64 kB) onto a SQS queue to process later in order. The Id-property needs to be unique and can be used in combination with the returned MessageId (see its result) into your own database/system. Using DelaySeconds makes it possible to hide a certain message a little bit longer (maximum of 900s = 15 minutes).

Example

var params = {
    QueueName   : 'yourQueueName',
    Id          : ['uniqueIdentifier1','uniqueIdentifier2'],
    MessageBody : [JSON.stringify({name:'Hello World',age:30}),JSON.stringify({name:'Bye Bye World',age:20})],
    DelaySeconds: [undefined,20]
};
sqs.SendMessageBatch(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
