GetQueueAttributes (Open docs on AWS)

Params

Action (required)
(param)
AttributeName (required)
(param-array)
QueueName (required)
(special)

Examples

List the attributes of a SQS queue.

This example shows how to list the attributes of a SQS queue. The AttributeName is uppercase-sensitive so "all" will not work, but "All" will!

Example

var params = {
    QueueName       : 'yourQueueName',
    AttributeName   : 'All'
}
sqs.GetQueueAttributes(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});

List a specific attribute of a SQS queue.

This example shows how to list a specific attribute (like when the queue was created) of a SQS queue. The AttributeName is uppercase-sensitive so "createdtimestamp" will not work, but "CreatedTimestamp" will! Use the array to add more attributes to embed in the response.

Example

var params = {
    QueueName       : 'yourQueueName',
    AttributeName   : ['CreatedTimestamp']
}
sqs.GetQueueAttributes(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
