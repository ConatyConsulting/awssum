ListBuckets (Open docs on AWS)

Params

Param Name Required Type Notes
This operation doesn't take any arguments.

Examples

List all buckets in a region

This example show how to list all buckets in a region. Keep in mind that bucket names need to be lowercase! S3 buckets live in 1 region so configuring your S3 endpoint will define in which region you check if your bucket exists.

Example

s3.ListBuckets(function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
s3.ListBuckets({}, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
