OptionsObject (Open docs on AWS)

Params

Param Name Required Type Notes
Origin required header
AccessControlRequestMethod required header
AccessControlRequestHeaders optional header

Examples

List if a call can be made (CORS)

This example show how to list if a call can be made from a specific origin (CORS). Keep in mind that bucket names need to be lowercase! S3 buckets live in 1 region so configuring your S3 endpoint will define in which region you check if your bucket exists.

Example

var params = {
    BucketName : 'somebucket',
    Origin : "http://www.domain.com",
    AccessControlRequestMethod : "GET"
};
s3.OptionsObject({}, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});

List if a call can be made (CORS) with headers

This example show how to list if a call can be made from a specific origin (CORS) with headers. Keep in mind that bucket names need to be lowercase! S3 buckets live in 1 region so configuring your S3 endpoint will define in which region you check if your bucket exists.

Example

var params = {
    BucketName : 'somebucket',
    Origin : "http://www.domain.com",
    AccessControlRequestMethod : "GET",
    AccessControlRequestHeaders : "Authorization, x-code-value"
};
s3.OptionsObject({}, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
