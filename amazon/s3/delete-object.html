Delete an object inside a S3 bucket

This example show how to delete an object in a S3 bucket. Keep in mind that bucket names need to be lowercase! S3 buckets live in 1 region so configuring your S3 endpoint will define in which region you check if your bucket exists.

Example

var params = {
    BucketName : 'somebucket',
    ObjectName : 'somefile.js'
};
s3.DeleteObject(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
