Examples

Delete a S3 bucket

This example show how to delete a S3 bucket. Keep in mind that bucket names need to be lowercase! S3 buckets live in 1 region so configuring your S3 endpoint will define which region your bucket will be made/deleted in.

Example

var params = {
    BucketName : 'somebucket'
};
s3.DeleteBucket(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
