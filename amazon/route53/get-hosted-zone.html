GetHostedZone (Open docs on AWS)

Params

Param Name Required Type Notes
HostedZoneId required special

Examples

Get information about a hosted zone

This example shows how to retrieve information about a hosted zone (like for example the nameservers).

Example

var params = {
    HostedZoneId : 'newdomain.com',
}

r53.GetHostedZone(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
