GetChange (Open docs on AWS)

Params

ChangeId (required)
(special)

Examples

Get information about a hosted zone

This example shows how to retrieve information about a hosted zone (like for example the nameservers).

Example

var params = {
    ChangeId : 'the-id-you-received-from-route53.ChangeResourceRecordSets()',
}

r53.GetChange(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
