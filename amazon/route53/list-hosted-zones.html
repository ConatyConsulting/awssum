ListHostedZones (Open docs on AWS)

Params

Param Name Required Type Notes
Marker optional param
MaxItems optional param

Examples

Show all hosted zones

This example shows how to list all hosted zones in your account. If you are not passing any parameters, then you may omit the param object.

Example

r53.ListHostedZones(function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});

r53.ListHostedZones({}, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});

Show a specific hosted zone

This example shows how to filter only one hosted zone in your request.

Example

var params = {
    HostedZone : '/hostedzone/asd78a6d987a68d',
}
r53.ListHostedZones(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
