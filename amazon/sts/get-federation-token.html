Get federation token for specific IAM user

This example shows how to get temporary security credentials for a specific IAM user. Your temporary credentials can last between 1 and 36 hours and are default set to 12 hours (DurationSeconds).

Example

var params = {
    Name            : 'uniquename',
    DurationSeconds : 50400
}
sqs.GetFederationToken(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});

Get federation token with a specific policy

This example shows how to get temporary security credentials with a specific policy. You will standard IAM policies and in this case it will allow all services on all resources. It will use the default duration (12 hours) while we omitted the DurationSeconds-property.

Example

var params = {
    Name    : 'uniquename',
    Policy  :'{"Statement": [{"Effect": "Allow","Action": "*","Resource": "*"}]}'
}
sqs.GetFederationToken(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
