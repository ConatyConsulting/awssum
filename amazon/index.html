EndPoints

All Amazon services require an endpoint when talking to them. However in many cases (e.g. SES), there is only one endpoint. In these cases, you do not need to provide it to the service constructor, instead it is assumed for you. In all of these cases, it is assumed to be US_EAST_1.

  • US_EAST_1
  • US_WEST_1
  • US_WEST_2
  • EU_WEST_1
  • AP_NORTHEAST_1
  • AP_SOUTHEAST_1
  • SA_EAST_1
  • US_GOV_WEST_1
