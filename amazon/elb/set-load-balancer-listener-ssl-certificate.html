Attach a ssl certificate on a loadbalancer

This example shows how to attach a ssl certificate on a loadbalancer. See more information at http://docs.amazonwebservices.com/IAM/latest/UserGuide/ManagingServerCerts.html

Example

var params = {
    SSLCertificateId    : 'sslCertificateIdentifier',
    LoadBalancerPort    : 80,
    LoadBalancerName    : 'test'
};
elb.SetLoadBalancerListenerSslCertificate(params,function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
