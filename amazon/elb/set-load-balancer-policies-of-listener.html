Updates policy of a loadbalancer listener

This example shows how to update policy of a loadbalancer listener. You can associate multiple policies with a listener.

Example

var params = {
    PolicyNames       : ['policyName1','policyName2'],
    LoadBalancerPort  : 80,
    LoadBalancerName  : 'test'
};
elb.SetLoadBalancerPoliciesOfListener(params,function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
