Register a or multiple instances with a loadbalancer

This example shows how to register an instance with a loadbalancer. Keep in mind that an instance can stick in "out of order" mode when it is initiated in a availability zone that the loadbalancer is not using. It will shows this inside the webconsole but will say "Could not find EC2 instance". It will also take some time before the status of the instance inside the loadbalancer is fully initiated. This is important to know if you want to quickly want to add instances.

Example

var params = {
    Instances           : [{InstanceId:'i-1234fake'}],
    LoadBalancerName    : 'test'
};
elb.RegisterInstancesWithLoadBalancer(params,function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
