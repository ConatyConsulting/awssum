Deregister a or multiple instances with a loadbalancer

This example shows how to deregister an instance with a loadbalancer. Keep in mind that you can only deregister instances that are running and fully connected to a loadbalancer, so "out of order" means it is not yet connected and cannot be deregistered!

Example

var params = {
    Instances           : [{InstanceId:'i-1234fake'}],
    LoadBalancerName    : 'test'
};
elb.DeregisterInstancesFromLoadBalancer(params,function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
