Examples

Define the expiration period of a cookie on a loadbalancer

This example shows how to define the expiration period of a cookie on a loadbalancer. It is only available with cookies through http/https.

Example

var params = {
    CookieExpirationPeriod  : 10000,
    PolicyName              : 'testPolicyExpiration',
    LoadBalancerName        : 'test'
};
elb.CreateLBCookieStickinessPolicy(params,function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
