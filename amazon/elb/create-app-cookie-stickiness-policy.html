Create an app cookie stickiness policy

This example shows how to create an app stickiness policy. It ensures stickiness with cookies through http/https.

Example

var params = {
    CookieName          : 'testCookie'
    PolicyName          : 'testPolicy',
    LoadBalancerName    : 'test'
};
elb.CreateAppCookieStickinessPolicy(params,function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
