Enable a healthcheck on a loadbalancer

This example shows how to enable a healthcheck on a loadbalancer. Ensure that you have a timeout that is less than the interval. In this case it will target through http port 80 (and you can add a path with folders behind it (see documentation)).

Example

var params = {
    Target              : 'http:80/',
    HealthyThreshold    : 10,
    Interval            : 60,
    Timeout             : 30,
    UnhealthyThreshold  : 3,
    LoadBalancerName    : 'test'
};
elb.ConfigureHealthCheck(params,function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
