Get identity DKIM attributes

This example shows how to get identity DKIM attributes to verify its DNS settings.

Example

var params = {
    Identities : [ 'email@domain.com', 'domain.com' ],
};
ses.GetIdentityDkimAttributes(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
