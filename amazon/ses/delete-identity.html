Examples

Delete an identity

This example shows how to delete an identity (e-mail or domain) regardless if it is verified or not. It replaces SES.DeleteVerifiedEmailAddress.

Example

ses.DeleteIdentity({Identity:'email@domain.com'},function(err,data){
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
ses.DeleteIdentity({Identity:'domain.com'},function(err,data){
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
