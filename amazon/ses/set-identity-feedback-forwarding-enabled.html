Examples

Set identity feedback forwarding to enabled/disabled

This example shows how to enable/disable feedback forwarding on a identity (domain/email address). It is important for bounce and complaint notifications.

Example

ses.SetIdentityFeedbackForwardingEnabled({Identity:'email@domain.com',ForwardingEnabled:true},function(err,data){
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
ses.SetIdentityFeedbackForwardingEnabled({Identity:'domain.com',ForwardingEnabled:false},function(err,data){
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
