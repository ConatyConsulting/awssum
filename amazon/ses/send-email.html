Examples

Send a simple email

This example shows how to send an e-mail without an attachment. Ensure you have verified your send e-mail address (and receive e-amil address if you are not in production mode).

Example

var params = {
    ToAddresses    : ['example@domain.com'],
    CcAddresses    : ['email@domain.com'],
    BccAddresses   : ['contact@domain.com'],
    Text           : 'This is an e-mail to test SES.\n\nThis is another line of text.',
    Html           : '<p>This is an e-mail to test SES.<br />This is another line of text',
    Subject        : 'Example SES mail',
    Source         : 'mail@anotherdomain.com'
};
ses.SendEmail(params,function(err,data){
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});

Send a simple email with specific charsets.

This example shows how to send an e-mail with a specific charset. Ensure you have verified your send e-mail address (and receive e-amil address if you are not in production mode).

Example

var params = {
    ToAddresses    : ['example@domain.com'],
    Text           : 'This is an e-mail to test SES with a smiley ☺.\n\nThis is another line of text.',
    TextCharset    : 'UTF-8',
    Html           : '<p>This is an e-mail to test SES with a smiley ☺.<br />This is another line of text',
    HtmlCharset    : 'UTF-8',
    Subject        : 'Example SES mail',
    SubjectCharset : 'UTF-8',
    Source         : 'mail@anotherdomain.com'
};
ses.SendEmail(params,function(err,data){
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
