Examples

Get identity notification attributes

This example shows how to get identity notification attributes for bounce and complaints notifications.

Example

var params = {
    Identities : [ 'email@domain.com', 'domain.com' ],
};
ses.GetIdentityNotificationAttributes(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
