Get identity verification attributes

This example shows how to get identity verification attributes for verification status/tokens.

Example

var params = {
    Identities : [ 'email@domain.com', 'domain.com' ],
};
ses.GetIdentityNotificationAttributes(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
