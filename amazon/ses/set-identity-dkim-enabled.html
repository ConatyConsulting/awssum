Examples

Set identity DKIM to enabled/disabled

This example shows how to enable/disable DKIM on a identity (domain/email address).

Example

ses.SetIdentityDkimEnabled({Identity:'email@domain.com',DkimEnabled:true},function(err,data){
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
ses.SetIdentityDkimEnabled({Identity:'domain.com',DkimEnabled:false},function(err,data){
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
