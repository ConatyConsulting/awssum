GetAttributes (Open docs on AWS)

Params

Param Name Required Type Notes
Action required param
DomainName required param
ItemName required param
AttributeName optional param-array
ConsistentRead optional param

Examples

Get a specific set of attributes from the item, rather than the entire item.

This example shows how to get the age and height attributes for a particular Item.

Example

var params = {
    DomainName : 'person',
    ItemName   : 'chilts',
    AttributeName : [
        'age', 'height'
    ],
};
sdb.GetAttributes(params, function(err, data) {
    fmt.msg("getting chilts - expecting success");
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
