Examples

Delete data of multiple items

This example shows how to delete specific attributes of multiple items. You can add a maximum of 25 item changes!

Example

// if "record1" or "record2" items exist with "name" and "age" properties, code below will delete only "name" property
var params = {
    DomainName      : 'an_unique_string',
    ItemName        : [
                        'record1',
                        'record2'
                      ],
    AttributeName   : [
                        ['name'],
                        ['name']
                      ],
    AttributeValue  : [
                        ['my_name'],
                        ['another_name']
                      ]
};
sdb.BatchDeleteAttributes(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
