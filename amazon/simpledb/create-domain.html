Create a domain

This example shows how to create a domain to send store messages in.

Example

var params = {
    DomainName : an_unique_string'
};
sdb.CreateDomain(params, function(err, data) {
    fmt.dump(err, 'Error');
    fmt.dump(data, 'Data');
});
