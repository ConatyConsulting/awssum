    Connect Middleware using AwsSum

    Connect and Express are brilliant, lovely and amazing tools. And we can harness the power of them and AwsSum inside some connect middleware. There are many things you could do, but here are a few already written for you.

    connect-stream-s3

    • Git Repo
    • npm

    This middleware lets you upload files directly from a form straight into Amazon S3. By using app.use(express.bodyParser());, we end up with a file on disk, which we can then stream into S3. By doing it this way, we can easily re-try if the S3 operation fails. Give it a go!

    connect-error-amazon-s3

    • Git Repo
    • npm

    This middleware lets you send connect/express errors to an email address. Generally you wouldn't use this for production but for testing or staging it could be a godsend.
