Installing

With NPM it's so easy to install AwsSum. Truly, the best package manager for any language.

{% highlight sh %} npm install awssum {% endhighlight %}

Or you can get a tarball from GitHub:

{% highlight sh %} wget https://github.com/appsattic/node-awssum/tarball/master {% endhighlight %}

Or clone the repo using Git:

{% highlight sh %} git clone git://github.com/appsattic/node-awssum.git {% endhighlight %} {% endcapture %} {% include main.html %}
