Installing

With NPM it's so easy to install AwsSum. Truly, the best package manager for any language. 

  $ npm install awssum

Or you can get a tarball from GitHub: 

  $ wget https://github.com/appsattic/node-awssum/tarball/master

Or clone the repo using Git: 

  $ git clone git://github.com/appsattic/node-awssum.git
