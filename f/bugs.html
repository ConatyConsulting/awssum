When submitting a bug report, it's essential and really useful to do the following:

  • check that this bug hasn't already been submitted
  • make sure your version of AwsSum is up to date (check you haven't pinned the version in your package.json)
  • include the node version number: node --version
  • include the awssum version number: cat ./node_modules/awssum/package.json
  • edit ./node_modules/awssum/lib/awssum.js and change var debug = false; to true

Please include all this info in the bug report, which you should enter here.

Many thanks. :)
