AwsSum is a Node.js library written by Andrew Chilton (Twitter). It is designed to be helpful in utilising many, extensive and sometimes quite different Web APIs in a consistent fashion. Which means that you don't have to deal with:

  • Request
    • which HTTP verb to use
    • web service endpoints
    • how to format your input parameters (URL params, XML/JSON bodies, forms, etc)
  • Response
    • HTTP Status Codes
    • getting other HTTP headers
    • decoding the returned body (whether XML, JSON, a file or nothing at all)

This is true of any web service you use through AwsSum.

It is designed as a low level wrapper around each of the operations each Web API defines. For example, Amazon EC2 has (currently) 134 operations, which means there is 134 operations defined in AwsSum.

Therefore, AwsSum is ideal for being built upon. See the following projects for where AwsSum is being utilised:

  • connect-stream-s3
  • connect-error-amazon-ses
  • winston-simpledb
  • winston-sns
  • jsbundle-s3
