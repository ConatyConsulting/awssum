Examples

This section is designed for you to see a few other ways of doing things with AwsSum. It is also to document some other packages which either compliment or build on AwsSum.

Complimentary Packages

  • oibackoff - an incremental backoff generator
    • Example usng oibackoff

Packages which use AwsSum

Since AwsSum is designed to be a one-to-one mapping of each WebAPI, it isn't always the friendliest way to use them. Instead, it is designed so that higher level packages can use it in a more friendly way. Here are some examples:

  • node-awssum-scripts - many example scripts including syncing to S3 and talking to CloudWatch
  • connect-stream-s3 - Streaming connect middleware for uploading files to Amazon S3
  • connect-error-amazon-ses - send connect errors to an email address using Amazon:SES

If you would like your project to appear here, please contact me via email or Twitter.
